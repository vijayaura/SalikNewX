# LIVA — AI Insurance Assistant

A futuristic, mobile-first single-page chat interface that condenses the full LIVA car insurance journey into **three clicks**.

## Journey (3 clicks)

| Click | Action | What happens |
|-------|--------|--------------|
| **1** | Show My Quote → Confirm Plan | AI presents pre-loaded vehicle data, recommended Comprehensive plan, and optional add-ons |
| **2** | Pay (after accepting T&C) | Summary review with pre-filled HSBC payment card |
| **3** | Verify & Issue Policy | Documents synced from vault → policy issued → download PDF |

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically `http://localhost:5173`).

## Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
